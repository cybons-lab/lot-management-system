
## 3. 材料ビジネス業務フロー - かんばんパターン⑤

納入先種別:デンソー製作所向け、注文形態:かんばん在庫品の業務フローです。

```mermaid
graph TB
    subgraph 得意先
        A1[オリジナル配信]
        A2[ロット取得]
        K1[かんばん]
        K2[かんばん<br/>原紙またはコピー]
    end
    
    subgraph セールスサポート
        B1[オリジナル取込]
        B2{エラー<br/>チェック}
        B3[マスタ修正等対処]
        B4[オリジナル更新]
        B5[オリジナル出力]
        B6[発注計画書]
        B7{発注確認}
        B8[ロット帳<br/>ロット取得]
        B9{ロット指定<br/>有無}
        B10[かんばん<br/>コピー]
    end
    
    subgraph 仕入元
    end
    
    subgraph 倉庫
        W1[出荷票]
        W2[かんばん<br/>原紙またはコピー]
    end
    
    subgraph SAP
        E1[(SAP)]
        E2[在庫一覧<br/>発注残]
        E3[(SAP<br/>発注入力、承認)]
        E4[ロット帳更新]
        E5[(SAP<br/>受注登録)]
        E6{●出荷票<br/>・出荷計算<br/>・個別出荷<br/>・出荷票FAX}
        E7[(SAP<br/>個別出荷)]
        E8[(SAP<br/>個別出荷)]
    end
    
    %% オリジナル配信からSAP登録まで
    A1 --> E1
    B1 <--> E1
    B1 --> B2
    B2 -->|なし| B3
    B2 -->|あり| B3
    B3 --> B4
    B4 --> B1
    
    %% 発注計画
    E1 --> E2
    E2 --> B5
    B5 --> B6
    B6 --> B7
    
    %% SAP発注とロット処理
    B7 --> E3
    E3 --> E4
    
    %% かんばん処理フロー
    K1 --> K2
    K2 --> B10
    B10 --> B9
    B9 -->|あり| B8
    B9 -->|なし| A2
    B8 --> A2
    
    %% 受注登録と出荷
    A2 --> E5
    E5 --> E6
    
    %% 出荷先の分岐
    E6 -->|出荷票<br/>出荷計算<br/>個別出荷<br/>出荷票FAX| E7
    E6 -->|個別出荷| E7
    
    %% 外部倉庫への出荷
    E7 -->|出荷計算:外部倉庫| W1
    E6 -->|出荷計算:中部LC等| W1
    E7 -->|個別出荷:中部LC等| W1
    
    %% 個別出荷
    E8 -->|個別出荷| W2
    E8 -->|外部倉庫へ<br/>出荷票FAX| End1[終了]
    
    style E1 fill:#a8d5e2
    style E3 fill:#a8d5e2
    style E5 fill:#a8d5e2
    style E6 fill:#a8d5e2
    style E7 fill:#a8d5e2
    style E8 fill:#a8d5e2
    style K1 fill:#d3d3d3
    style K2 fill:#d3d3d3
    style W1 fill:#f0e68c
    style W2 fill:#f0e68c
```

## レビューポイント

このドキュメントをレビューされる際は、以下の点をご確認ください:

1. **図の可読性**: マーメイド図が複雑すぎないか、適切に分割されているか
2. **フロー図の論理性**: 各ステップの流れが論理的で理解しやすいか
3. **用語の一貫性**: 同じ概念に対して異なる用語が使われていないか
4. **色分けやスタイル**: 視覚的に区別しやすく、意味のある色分けになっているか
5. **省略可能な要素**: 必須ではない説明やステップが含まれていないか
6. **不足している情報**: 理解に必要な説明が欠けていないか

改善提案がありましたら、具体的な箇所と修正案をご指摘ください。
