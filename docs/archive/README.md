# Archive - Lot Management System

このディレクトリには、過去のドキュメント・設計書・不採用の仕様書などを保管しています。

## Archive ルール

1. **削除禁止**: アーカイブ内のファイルは削除せず保管します
2. **編集禁止**: アーカイブ内のファイルは編集しません（参照のみ）
3. **参照のみ**: 過去の経緯や設計判断を振り返るために使用します

## アーカイブ対象

- 旧バージョンのドキュメント
- 不採用の設計書
- 試作版・ドラフト版の仕様書
- 廃止された API ドキュメント
- 移行前の設計資料
